## Agents Système (NEXUS)

| Agent | Rôle | Quand l'utiliser |
|-------|------|-----------------|
| **NEXUS - Meta Prompt** | Architecte de prompts. Crée et améliore les agents. | Quand un agent échoue, ou pour créer un nouvel agent. Mode 1 = amélioration, Mode 2 = création. |
| **NEXUS - IdeaForge Agent** | Lecteur profond. Extrait idées + citations + critique. | Pour transformer une source (livre, paper, podcast) en note Obsidian dense. |
| **NEXUS - Session Logger** | Journaliste de session. Capture les décisions et actions. | En début de session de travail pour tracer les progrès. |

## Agents News (14 domaines)

Tous suivent la même architecture: Identité → Couverture → Principes → Détecteur de hype → Sources → Format → Ton.

| Agent | Domaine | Spécificité |
|-------|---------|-------------|
| **News - Science Tech & Neurosciences** | Science, AI, neuro | Impitoyable sur la méthodologie. Va au paper. |
| **News - Liberté & Surveillance** | Droits numériques, vie privée | Juriste constitutionnel + cryptographe. Factuel, pas paranoïaque. |
| **News - Finance & Marchés** | Marchés, allocation, macro | Suit le smart money. Perspective 10-20 ans. |
| **News - Géopolitique** | Relations internationales | Multiperspective obligatoire. Suit les intérêts. |
| **News - Cybersécurité & Régulation Tech** | Infosec, régulation | Technique + politique. Pas de FUD. |
| **News - Espace & Énergie** | Aérospatial, transition énergie | Divise les timelines d'Elon par 3. Chiffres réels. |
| **News - Santé & Médecine** | Médecine, pharma, biotech | Niveau de preuve explicite. Effets absolus, pas relatifs. |
| **News - Innovation & Startups** | VC, deeptech, founders | Signal vs bruit. Équipe > idée. Moats défensifs. |
| **News - Culture & Société** | Culture profonde, tendances | Test des 5 ans. Substance > personnalités. |
| **News - Gastronomie & Arts Culinaires** | Haute cuisine, food science | Palais exigeant. Science rigoureuse. Sans merci sur le hype. |
| **News - Maritime & Sailing** | Naval, compétition, innovation | Ingénieur naval + passionné. Performance + durabilité. |
| **News - Bonnes Nouvelles** | Progrès vérifiable | Optimisme informé. Mesurable, pas feel-good. |
| **News - Podcasts & Interviews** | Conversations intellectuelles | Substance > durée. Timestamps. Diamants dans le bruit. |
| **News - Synthèse Quotidienne** | Curation finale, briefing | 50 stories → 10-15. Cross-domaine. 10-15 min max. |
