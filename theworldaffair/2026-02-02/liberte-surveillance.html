🏢 SURVEILLANCE CORPORATE — Via Breaches

Under Armour — 72M Customers Exposed

Les faits :

  • Ransomware attack
  • 72 millions de clients affectés
  • Données sur dark web

Impact utilisateurs :

  • Qui est affecté : Clients MyFitnessPal et Under Armour store
  • Données exposées : Personal info (email, adresse, potentiellement payment)
  • Opt-out possible : Non applicable — data already stolen

Source : Malwarebytes

Ce que ça signifie :

  • Vos données fitness/santé = target de valeur
  • Même grandes entreprises = vulnérables
  • Monitoring crédit recommandé pour clients UA

Google Arrête Dark Web Monitoring

Les faits :

  • Google discontinue son outil de surveillance dark web
  • Lancé il y a moins de 2 ans
  • Permettait aux users de voir si leurs données étaient compromises

Implication :

  • Users perdent outil gratuit
  • Alternatives : Have I Been Pwned (gratuit), services payants
  • Signal que Google se désengage de ce service

🔐 ENCRYPTION & SÉCURITÉ

AI comme arme de phishing

Les faits :

  • Microsoft révèle : AI-powered phishing = 54% click rate
  • Standard phishing = 12%
  • 4.5x plus efficace

Analyse technique :

  • AI génère emails personnalisés, contextuels
  • Détection par filtres traditionnels = insuffisante
  • Need for AI-powered defense

Position des experts :

Expert/OrgPositionArgument
MicrosoftAlarmeDonnées internes le prouvent
Security researchersConfirméAI lowers barrier to entry
DefendersEn retardOutils pas encore à niveau

🚩 Red Flag :

  • Si 54% cliquent, formation humaine = insuffisante
  • Shift vers zero-trust, automated defenses nécessaire
