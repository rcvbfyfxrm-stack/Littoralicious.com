🔴 ALERTES CRITIQUES

🛡️ CYBERSÉCURITÉ — INCIDENTS & MENACES

Under Armour — Ransomware Breach 72M Customers

Les faits :

  • Données de 72 millions de clients sur le dark web
  • Type d'attaque : Ransomware
  • Données compromises : Informations personnelles clients

Source primaire : Malwarebytes — Under Armour Breach

Analyse :

  • Une des plus grosses breaches retail de 2026
  • Under Armour était déjà victime d'une breach en 2018 (MyFitnessPal)
  • Pattern : même entreprise ciblée plusieurs fois

Brightspeed — 1M+ Customers Claimed

Les faits :

  • US fiber broadband provider
  • Hacking group "Crimson Collective" revendique l'attaque
  • Claim : 1M+ customer personal information exfiltrée
  • Brightspeed "investigating"

Source primaire : SecurityWeek — Brightspeed

Évaluation :

  • Confiance dans claim : MOYENNE (group moins connu)
  • Telecom = infrastructure critique
  • À surveiller : Confirmation ou démenti Brightspeed

Illinois DHS — 700,000 Personnes Affectées

Les faits :

  • Illinois Department of Human Services breach
  • 32,000 Division of Rehabilitation Services customers
  • 672,000 Medicaid/Medicare recipients
  • Données : Personal + medical information

Source primaire : SecurityWeek — IDHS Breach

Implications :

  • Healthcare data = high value pour criminals
  • Government agencies = targets persistants
  • 700K = breach majeure santé publique

AI-Powered Phishing — Microsoft Warning

Les faits :

  • Microsoft révèle que adversaries utilisent AI pour phishing
  • 54% click-through rate (vs 12% standard)
  • 4.5x increase en efficacité

Source primaire : The Hacker News — AI Phishing

Implications :

  • AI as weapon devient mainstream
  • Défenses traditionnelles (training, filters) insuffisantes
  • Need for AI-powered defenses urgent

🚩 Red Flag :

  • Si 54% click sur phishing AI, l'humain n'est plus la défense viable

Chrome Extensions Malveillantes — 5 Nouvelles

Les faits :

  • 5 extensions malveillantes découvertes
  • Se font passer pour : Workday, NetSuite, SuccessFactors
  • Objectif : Steal authentication tokens, session hijacking
  • Technique : Block incident response capabilities

Source primaire : The Hacker News

Défenses recommandées :

MesurePrioritéDifficulté
Audit extensions installéesCritiqueFacile
Whitelist corporate extensions onlyHauteMoyenne
Browser isolation pour apps sensiblesHauteÉlevée

💰 RANSOMWARE & CYBERCRIME

Stats 2025 — Emsisoft Report

Chiffres clés :

  • 8,000+ organisations ciblées en 2025
  • +33% vs 2024 (~6,000)
  • +30% de groupes ransomware actifs

Tendance :

  • Ransomware industrialisé et en croissance
  • Plus de groupes = plus de compétition = plus d'attaques
  • Defense not keeping pace

⚖️ RÉGULATION — TECH

Google Discontinue Dark Web Report Tool

Les faits :

  • Google arrête son dark web monitoring tool
  • Lancé il y a moins de 2 ans
  • Permettait de voir si personal info était sur dark web

Implication :

  • Users perdent un outil gratuit de surveillance
  • Alternatives payantes restent (Have I Been Pwned, etc.)
