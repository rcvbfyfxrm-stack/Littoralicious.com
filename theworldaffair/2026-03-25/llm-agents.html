Claude prend les commandes : Anthropic lance le pilotage autonome d'ordinateur

Mercredi 25 Mars 2026 — Edition N°57

Claude Computer Use : l'agent qui pilote votre ecran

Anthropic a annonce le 24 mars que Claude peut desormais utiliser un ordinateur pour accomplir des taches de bout en bout. L'utilisateur envoie une demande depuis son telephone, et Claude ouvre des applications, navigue dans un navigateur web, remplit des tableurs -- tout ce qu'un humain ferait assis a son bureau. Le mode 'auto' de Claude Code, en research preview sur macOS, permet a l'IA de decider quelles actions sont suffisamment sures pour etre executees sans validation humaine, avec un systeme de verification integre qui detecte les comportements risques et les injections de prompt. (Source: CNBC)

Claude depasse ChatGPT dans l'App Store

Le contexte est explosif : Claude a depasse ChatGPT au classement de l'App Store debut mars apres que les utilisateurs ont boycotte OpenAI suite a son contrat de 200 millions de dollars avec le Pentagone. Le revenu annualise de Claude Code seul depasse 2,5 milliards de dollars en fevrier 2026, mettant Anthropic en position de depasser le CA total d'OpenAI d'ici fin 2026. (Source: Fortune)

Oracle AI Database 26ai et agents en base de donnees

Oracle a devoile le 24 mars sa suite d'innovations agentic pour l'Oracle AI Database 26ai, incluant l'Unified Memory Core (memoire persistante pour agents IA dans le moteur de base de donnees) et l'AI Database Private Agent Factory, une plateforme no-code pour deployer des agents en conteneurs portables. L'objectif : positionner la base de donnees comme couche d'orchestration centrale pour les workflows multi-agents. (Source: Futurum Group)

Google TurboQuant : les LLM sur votre telephone

Google a publie TurboQuant, une technique de compression qui reduit la memoire des LLM par 6x et accelere l'inference jusqu'a 8x sans perte de precision mesurable. Des modeles performants peuvent desormais tourner localement sur un Mac Mini 16 Go ou un smartphone haut de gamme. (Source: LLM Stats)

Signaux Faibles

  • Le passage de Claude au No.1 de l'App Store sur fond de boycott anti-Pentagone montre que les convictions ethiques des utilisateurs deviennent un facteur de marche determinant dans l'IA.
  • L'approche Oracle (agents dans la base de donnees) signale un mouvement vers l'infrastructure agentic invisible, en opposition au modele chatbot visible.
  • La compression TurboQuant de Google pourrait democratiser l'inference locale plus vite que prevu, reduisant la dependance aux API cloud.
