Newsletter (Buttondown)

Your Buttondown account: buttondown.com/littoralicious

Admin dashboard: buttondown.com/emails — see subscribers, open rates, and send emails.

Subscribers page: buttondown.com/subscribers — see who signed up, when, and their status.

RSS-to-email: Connect your RSS feed to auto-notify subscribers when you publish. Go to Settings → Newsletter → RSS feed URL: https://www.littoralicious.com/feed.xml

API key: If you want programmatic access, get it from Settings → API. The API endpoint is https://api.buttondown.com/v1/

Welcome email: Go to Settings → Emails → enable "Send welcome email" and paste the welcome text below.

Setup Notes

Firestore Rules: Make sure the pageviews and stats collections allow writes from the client. In Firebase Console → Firestore → Rules, ensure these collections have write access:

match /pageviews/{doc} { allow create: if true; allow read: if true; }

match /stats/{doc} { allow write: if true; allow read: if true; }

The articles collection should already have read/write access from the community features.
